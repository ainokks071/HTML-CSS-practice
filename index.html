HTML과 CSS를 이용하여 홈페이지 만들기

* HTML 기본태그 6가지

1. 텍스트 태그

h1 tag : 제목

h2 tag : 부제목

p tag : 본문

2. 미디어 태그

img tag : 이미지

방법 : 구글 이미지 우클릭 -> 이미지 주소 복사 -> img의 src

3. 링크 태그

a href tag : 링크 (href : 연결할 주소)

이미지 클릭

클릭 : Naver (새창열기 : blank)

4. 리스트 태그

ul(unordered list), ol(ordered list), li(list item) tag : 리스트

[프론트엔드 언어의 종류]
  • HTML
  • CSS
  • JavaScript
  1. HTML
  2. CSS
  3. JavaScript

5. 테이블 태그

table, tr, td tag : 표

[프론트엔드 언어의 역할]
언어 역할
HTML 문서의 뼈대
CSS 문서 꾸미기
JS 문서에 기능 부여

6. 입력 태그

input tag, textarea tag : 텍스트 입력 기능
